**Supply Chain Efficiency & Inventory Optimization Dashboard**

This dashboard helps businesses reduce stockouts, optimize inventory, and improve supplier performance using data-driven insights.

**Project Overview**

This project is an end-to-end Power BI dashboard designed to analyze supply chain performance, optimize inventory levels, and support data-driven decision-making.

It covers inventory tracking, supplier performance, demand forecasting, and advanced optimization techniques like EOQ, Safety Stock, and Reorder Point.


**Tools Used**

* Power BI
* DAX
* SQL
* Excel


**Dataset Description**

The dataset includes:

* Product details
* Inventory levels (opening, closing stock)
* Supplier performance
* Lead time & delivery status
* Demand forecast vs actual demand
* Inventory cost


**Key KPIs**

* Total Inventory
* Total Units Sold
* Stockout Count
* On-Time Delivery %
* Forecast Accuracy %
* Inventory Turnover Ratio


**Advanced Analytics**

* Economic Order Quantity (EOQ)
* Safety Stock
* Reorder Point (ROP)
* Stock Gap Analysis


**Dashboard Features**

* Inventory analysis by product and category
* Supplier performance tracking
* Demand vs forecast comparison
* Stockout and reorder monitoring
* Interactive slicers for dynamic filtering


**Key Insights**

* Identified understocking risk across products
* Highlighted supplier delays affecting inventory
* Detected forecasting inefficiencies
* Recommended improvements in reorder policies
