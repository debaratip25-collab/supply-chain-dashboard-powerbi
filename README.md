# Supply Chain Efficiency & Inventory Optimization Dashboard

## Project Overview

This project is an end-to-end Power BI dashboard designed to analyze supply chain performance, monitor inventory levels, and optimize ordering decisions using data-driven insights.

The dashboard integrates demand analysis, supplier performance, inventory tracking, and advanced optimization techniques such as Economic Order Quantity (EOQ), Safety Stock, and Reorder Point (ROP).

---

## Business Objective

The objective of this project is to help organizations:

* Reduce stockouts and lost sales
* Optimize inventory holding costs
* Improve supplier performance
* Enhance demand forecasting accuracy
* Enable data-driven supply chain decision-making

---

## Tools and Technologies

* Power BI
* DAX
* SQL
* Excel

---

## Dataset Description

The dataset includes:

* Product and category details
* Inventory levels (opening, incoming, and closing stock)
* Supplier and delivery performance
* Lead time and delivery delays
* Demand forecast versus actual demand
* Inventory cost and turnover metrics

---

## Key Performance Indicators

* Total Inventory
* Total Units Sold
* Stockout Count
* On-Time Delivery Percentage
* Forecast Accuracy Percentage
* Inventory Turnover Ratio

---

## Advanced Features

* Economic Order Quantity (EOQ) calculation
* Safety Stock estimation
* Reorder Point (ROP) modeling
* Stock Gap analysis for risk identification
* What-if simulation using demand growth and lead time adjustments

---

## Dashboard Features

* Inventory analysis by product and category
* Supplier performance monitoring
* Demand versus forecast comparison
* Stockout and reorder analysis
* Interactive slicers for filtering by region, product, supplier, and time
* Scenario-based analysis using what-if parameters

---

## Key Insights

* Identified understocking risks due to lower reorder levels compared to calculated reorder points
* Highlighted supplier delays affecting inventory availability and planning
* Detected demand variability influencing safety stock requirements
* Revealed forecasting inefficiencies leading to both overstocking and stock shortages
* Demonstrated how changes in demand and lead time significantly impact reorder decisions
