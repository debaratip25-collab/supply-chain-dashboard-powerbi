# Supply Chain Efficiency & Inventory Optimization Dashboard

## 📊 Project Overview

This project is an end-to-end Power BI dashboard designed to analyze supply chain performance, monitor inventory, and optimize ordering decisions.

## 🛠 Tools Used

* Power BI
* DAX
* SQL
* Excel

## 📦 Key Features

* Inventory analysis (opening, closing, turnover)
* Supplier performance tracking
* Demand vs forecast comparison
* Stockout and reorder analysis
* EOQ, Safety Stock, and Reorder Point calculation

## 📈 KPIs

* Total Inventory
* Total Units Sold
* Stockout Count
* On-Time Delivery %
* Forecast Accuracy %
* Inventory Turnover Ratio

## 🔍 Key Insights

* Identified understocking risk across products
* Highlighted supplier delays
* Detected demand forecasting gaps
* Provided inventory optimization recommendations
